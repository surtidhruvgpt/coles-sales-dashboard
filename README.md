# Coles Supermarket Sales Dashboard

## 📊 Project Overview
Interactive Power BI dashboard analyzing Coles supermarket sales performance across Australian states. Provides insights into store revenue, performance metrics, and cost analysis to support business decision-making.

## 🎯 Key Insights
- **Total Gross Sales:** $28,577M across all stores
- **Avg Revenue per Store:** $33M
- **Top Performing State:** NSW (28.25% of total revenue)
- **Staff Productivity:** $2M revenue per staff member
- **Performance Status:** Tracks stores meeting/missing revenue targets by quarter

## 📈 Dashboard Features

### KPI Cards
- Total Gross Sales ($M)
- Avg Revenue per Store ($M)
- Avg Revenue per Staff ($M)

### Visualizations
- **Revenue Distribution by State (Donut):** Shows revenue contribution by state (VIC, NSW, QLD, WA, SA, ACT)
- **Expected vs Actual Sales (Column Chart):** Compares expected revenue against actual gross sales by quarter
- **Sales Cost by State (Bar Chart):** Identifies cost burden across states
- **Store Performance Table:** Detailed metrics including store ID, location, gross sales, forecast status, and efficiency ratios

### Interactive Filters
- Filter by Quarter (Q1 2023, Q2 2023)
- Filter by Performance Status (On Target, Below Target)

## 🛠️ Tools & Technologies
- **Power BI Desktop** — Dashboard creation & visualization
- **Power Query** — Data cleaning & transformation

## 📊 Data Source
- **ColesStoreData.csv** — Store metadata (location, staff count, store area, customer count)
- **ColesSalesData.csv** — Sales performance data (revenue, costs, targets, quarterly performance)

## 🔑 Key Metrics
- **Avg Revenue per Staff:** $2M (staff productivity measure)
- **Revenue per SqM:** Efficiency of store space utilization
- **Avg Customer Value:** Revenue generated per customer
- **Cost as % of Revenue:** Cost efficiency ratio
